## Why these visuals

The three screenshots above document that APERTURE is a **working
operational system at corpus scale**, not a paper-only artifact. The
Pipeline Dashboard image shows reviewers the per-book operational
view of the full pipeline; the two Character Network images show
how that evidence collapses to a single character, first at
the radar-profile level (Jane) and then at the waveform-trajectory
level (Elizabeth).

The four `f*.pdf` figures are publication-quality figures referenced
from the paper. They are regenerated by the corresponding
reproduction scripts and notebooks; the PDFs in this folder are the
canonical deliverables for the paper.

## Adding new figures

1. Drop the PDF or PNG in this directory.
2. Add a row to the appropriate table above with file, subject,
   source.
3. Reference it from the reproduction table in the repo-root `README.md`
   if it is a paper-claim artifact.
